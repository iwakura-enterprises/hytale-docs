# Welcome to Voile!

<gradient data="red:yellow:green">Thank you for installing Voile!</gradient>

Voile is a **documentation framework** allowing server owners and mod developers to create **user-friendly wiki**
accessible right from the game. Within this internal wiki, you will learn how to use Voile.

> This documentation is available on the web as well: **https://docs.voile.dev**
