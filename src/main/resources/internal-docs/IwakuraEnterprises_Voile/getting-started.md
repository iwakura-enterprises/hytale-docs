# 1. Downloading & installing

Installing Voile should be as simple as downloading its JAR file and dropping it into the corresponding mods folder.

## Dedicated server

1. Download Voile at &9&lhttps://download.voile.dev
2. Move the downloaded JAR file into the mods directory
3. Restart the server
4. Run `/voile` command

! You may need to add the corresponding permission to use the `/voile` command.
! This can be done using Hytale's permission command.
! 
! ```
! /perm group add Adventure iwakuraenterprises.voile.command.voile
! ```

## Single-player world

1. Download Voile at &9&lhttps://download.voile.dev
2. Right-click your world
3. Click **OPEN FOLDER** button
4. Move the downloaded JAR file into the mods directory
5. Click the **MODS** tab next to **SETTINGS** and **BACKUPS**
6. Verify that Voile is enabled by checking its checkbox on the right side
7. Enter your world
8. Run `/voile` command

! You may need OP to use the `/voile` command.
! 
! ```
! /op self 
! ```
